Внутренняя статистика выполнения задач

У нас появилась таблица для ведения задач. Теперь не нужно скидывать Артему на почту список задач. Каждый верстальщик отмечает новополученые, и сделаные задачи в этой таблице. Таблица делится на 2 части:

  • Активные задачи с прошлой недели (заполняется Артемом Сытником(сюда входят все незавершенные задачи с прошлой недели)) и
  • Полученные задачи на этой неделе (заполняет каждый верстальщик, сразу полсе получения задачи. Все задачи что получаете на текущей неделе)

В колонке исполнитель верстальщик отмечает себя рядом со ссылкой на свою задачу

После выполнения своей задачи верстальщик отмечает в таблице свою строку с задачей зеленым цветом(задачи где учавствует несколько верстальщиков отмечает зеленым Артем/Сергей)

Далее Артем ставит статус(сделано/перенесено на след неделю). Задачи со статусом "сделано" вносятся в еженедельный отчет в графу "Завершенные задачи". Задачи со статусом "перенесено на след неделю" вносятся в еженедельный отчет в графу "В работе".

Поле комментарий предназначено для запроса данных/описания проблемы/обьяснения зависания задачи/уточнения

Так же в таблице есть вкладки. Одна вкладка под одну неделю, прошедьшие недели будут скрывтаься.

Статистика сверстаных страниц

Статстика представляет собой таблицу, для каждого верстальщика есть вкладка с фамилией верстальщика. В своей вкладке верстальщик ежедневно фиксирует количество сверстаных страниц. После пятого дня добавляется общая колонка, в которой идут подсчеты всех страниц. Важно отмечать корректные данные в таблице.

После выполнения задачи верстальщик заливает название задачи зелёным цветом, а Артём при сборе в конце отчётной недели (вторник, 12:00), проставляет отметку "+" о том что задача попала в список выполненых задач http://prntscr.com/jrzow6. Задачи которые будут залитые зелёным после вторника 12:00 и незалитые задачи будут перенесенны на следуйщую неделю в список активных задач (с сохранением заливки).
